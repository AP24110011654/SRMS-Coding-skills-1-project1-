# Student Information Management System in C

This project is designed to maintain student records using a menu-based system.  
It is developed for the **Coding Skills - 1** course at **SRM University, AP**.

The purpose of this project is to provide a simple storage and retrieval system for:
- Roll Numbers
- Student Names
- Marks

All operations are performed directly on text files for permanent storage.
